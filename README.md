This is the simulation files for the research paper "Hierarchical Communication Structures for Collective Decision Making in Robot Swarms" by Kevin Wu.
All Webots simulation files including code and extracted data can be found in this repo.
While Webots itself may be installed at:
https://cyberbotics.com/
The project can be accessed by opening the “Swarm decision making” folder and opening the file “emitterandreceivertesting_ver2.wbt” in Webots. It is recommended that for optimal performance, the graphics settings are all turned to the lowest.
To access the simulation open the .wbt file.
